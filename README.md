# distributed-scrapper
Microservices based distributed scrapper for fetching categorized Wikipedia content. Used for testing mircorservices intresting solutions.
Used:
	- Go
	- Kafka
	- RabbitMQ
	- Istio
	- Darp
	- CocroachDB
	- NoSQL (to choose)
